# AIO devcontainer feature

Install basic devtools and tmux/neovim/zsh with basic configure.

## Example Usage

All default value is true

```json
{
  "features": {
    "git@github.com:zqqqqz2000/devcontainer-features.git": {
      "enable_neovim": false,
      "enable_zsh": false,
      "enable_tmux": false
    }
  }
}
```
